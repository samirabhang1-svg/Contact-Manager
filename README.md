📒 Contact Manager - Spring Boot CRUD Application

«A simple Contact Manager web application built while learning Spring Boot, Spring MVC, Spring Data JPA, and Thymeleaf. This project implements complete CRUD operations and demonstrates the MVC architecture, request flow, and database integration.»

---

📌 Overview

The Contact Manager application allows users to create, view, update, and delete contacts through a clean web interface. It follows a layered architecture using Controller, Service, Repository, and Entity classes to separate responsibilities and improve maintainability.

---

✨ Features

- ➕ Add New Contact
- 📋 View All Contacts
- ✏️ Update Existing Contact
- 🗑️ Delete Contact
- 🎨 Simple and Responsive UI
- 🗄️ MySQL Database Integration
- 🏗️ Spring MVC Architecture
